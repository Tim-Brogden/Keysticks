KEYSTICKS
Getting started

Thank you for downloading Keysticks! Keysticks is a program that lets you control your Windows computer using Xbox 360 controllers, Xbox One controllers, joysticks and other game controllers.

There are lots of ways to use Keysticks. For example, you can...

  • Create your own controls for PC games...
  • Change the music playing on your laptop...
  • Browse the web on your TV from the sofa...

When Keysticks is running, it will automatically detect your controller when you plug it in. (If your controller isn't detected, check in the Windows Device Manager that you have the right drivers for it installed.) If you want to use an Xbox 360 controller, you can use either a wired controller plugged directly into a USB port, or a wireless controller with a Wireless Gaming Receiver plugged into a USB port (see picture below).

Once Keysticks is installed, you can start the program by:

  • Double-clicking the shortcut on your desktop
  • Clicking the shortcut in your Start menu, under All Programs/Keysticks
  • Double-clicking a Keysticks profile file

A "profile" is just a file that defines some controls for your controller(s); in other words, what you want the controller(s) to do. Keysticks comes with some ready-made profiles, and when you run Keysticks for the first time, it will load profile P01 to get you started. You can also download more profiles that other users have shared from within the software. It's useful to know that you can double-click profile files when Keysticks is already running, as a way of quickly loading a different profile.

When Keysticks is running, you'll see the controller window which is shaped like a game controller and shows you what the current controls are. This window has five menu buttons which, from left to right, are:

  • Choose a profile
  • Edit this profile
  • Open the program menu (also pictured below)
  • Program options
  • Open this help file

You can also access the program menu by right-clicking the controller image. You can close the controller window if you wish, once you're familiar with the controls: Keysticks will continue running in the background and you can access the program menu by clicking the icon in the system tray (bottom-right of your screen). You may have to click the arrow to unhide any hidden icons. To exit Keysticks, choose Exit from the program menu (see above).

Before we continue, it's worth mentioning that there's a Support area at Keysticks.net where you can find the online help, FAQ, and more. Please take note of the following caution too:

Take care while you are familiarising yourself with Keysticks controls, especially more powerful ones like Backspace, Delete, double-click, Start program, etc. It's a good idea to practise using a program like WordPad or a single-player game until you get the hang of things. Please follow the manufacturer's guidelines for using your controller (e.g. take regular rest breaks). If you plan to try Keysticks for health reasons, that's great to hear, but please seek medical advice first.
